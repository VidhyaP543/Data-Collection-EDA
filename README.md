# Data Collection, Cleaning & Exploratory Data Analysis

## Objective

Perform data cleaning and exploratory data analysis on the Titanic dataset.

## Dataset

Titanic Dataset (CSV)

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Data Cleaning

- Filled missing Age values
- Filled missing Embarked values
- Removed Cabin column
- Removed duplicate rows

## Exploratory Data Analysis

- Survival Count
- Gender Distribution
- Age Distribution
- Passenger Class
- Survival by Gender
- Correlation Heatmap

## Key Findings

- Females had a higher survival rate.
- Most passengers were aged 20–40.
- Third-class passengers were the largest group.
- Missing values were successfully handled.

##Key Observations
1. Female passengers survived more than males.

2. Most passengers were between 20–40 years old.

3. Passenger Class 3 had the highest number of passengers.

4. Age had missing values which were replaced with the median.

5. Cabin column contained many missing values and was removed.
